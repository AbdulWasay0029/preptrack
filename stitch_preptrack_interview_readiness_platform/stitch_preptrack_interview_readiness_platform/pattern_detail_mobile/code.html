Sliding Window

lightbulb

When to use this pattern

This technique is primarily used for solving problems that involve arrays or lists where you need to find a subset of data that meets certain criteria.

It is ideal when you are asked to find the longest/shortest substring, subarray, or a specific sum within a linear data structure.

Use it to convert a nested loop O(N²) solution into an optimized linear O(N) solution by maintaining a window that expands or shrinks.

Perfect for "contiguous" element problems where the window slides over the input rather than recalculating from scratch for every index.
